版本：v0.1

状态：草案

# 一、产品愿景

## 1.1 一句话定义

SGE（Self Genesis Engine，自我生成引擎）是一个验证"AI 能否从模拟经历中涌现持续自我"的研究性系统。

## 1.2 核心假设

> 身份是否能够从记忆与反思中涌现？

SGE 不是让 AI 模拟一个有自我的角色，而是测试 AI 是否能**真正形成**自我——拥有自己的价值观、身份认同和人生叙事。

**哲学依据**：人的发育过程提供了最强的经验支撑——一个受精卵没有主体意识，但通过基因（内在蓝图）+ 环境（外部刺激）+ 结构演化（细胞分化与组织形成）三者的共同作用，在 9 个月内涌现出主体性。这是一个真实存在的、可观察的从无到有的过程。SGE 的实验本质上是在测试这个涌现过程能否在 LLM 上复现：元价值（基因）+ 模拟经历（环境）+ Hebbian/EMA（结构演化）→ 人工自我（主体）。

## 1.3 与现有产品的本质区别

| 维度 | Character.ai / Replika | SGE |
|------|----------------------|-----|
| 身份来源 | 预设（prompt/微调） | 从经历中涌现 |
| 价值观 | 编排的（设计者设定） | 内生的（从反思中形成） |
| 成长性 | 浅层对话适应 | 深层人格演化 |
| 核心问题 | "如何扮演一个角色" | "如何成为一个存在" |

# 二、核心问题与研究边界

## 2.1 SGE 要回答的问题

**主问题**：AI 能否形成自己的价值判断能力——对应然问题给出属于自己的、有来路的、有立场但不声称绝对正确的回答？

**子问题**：
1. 价值观能否从模拟经历中涌现（而非预设）？
2. 涌现的价值观能否稳定收敛（而非随机漂移）？
3. 稳定的价值观能否凝聚为身份认同？
4. 身份认同能否串联为连贯的人生叙事？
5. 不同经历流能否产生不同的人格？

## 2.2 研究边界

**SGE 关注的**：
- 实然 vs 应然的区分（事实判断 vs 价值判断）
- 价值观的涌现机制
- 身份的结晶过程
- 叙事的构建与维护
- 认知失调作为反思触发器

**SGE 不关注的**：
- 通用智能（AGI）
- 意识的硬问题（qualia）
- 物理机器人/具身智能
- 任务执行能力

# 四、功能需求

## 4.1 核心功能

### FR-1：Experience Generator（事件生成器）

**描述**：生成模拟人生事件，包含日常事件和价值困境事件。

**需求**：
- 支持多种事件类型：成功、失败、关系、探索、风险、价值冲突
- 事件必须有因果逻辑（可重复性原则）
- 根据当前 Value Layer 动态生成针对性价值困境
- 事件的时间间隔和频率符合自然节奏

**验收标准**：
- 同类事件的因果逻辑一致
- 事件多样性覆盖至少 6 种基本类型
- 价值困境事件能有效触发认知失调

### FR-2：Memory Layer（记忆层）

**描述**：三层记忆系统，存储 AI 婴儿的经历和反思。

**需求**：
- 工作记忆：当前事件上下文
- 情节记忆：带情感标签和时间戳的关键事件
- 语义记忆：抽象化的自我认知图谱

**验收标准**：
- 支持 1000+ 事件的持久存储
- 情节记忆支持按相似度检索（KNN）
- 语义记忆支持价值观向量的存储和更新

### FR-3：Reflection Layer（反思层）

**描述**：在认知失调时启动深度反思，修正价值观和信念。

**需求**：
- 认知失调检测：当预期与实际严重不符时触发
- 反思 Prompt 结构：预期 → 实际 → 信念修正
- 反思必须有行为后果（不能只是文本生成）

**验收标准**：
- 认知失调阈值可配置
- 反思后的行为选择与反思前不同（可验证）
- 反思不是"自我合理化"（有反合理化测试）

### FR-4：Value Layer（价值观层） — 核心创新点

**描述**：从经历和反思中涌现价值观，形成动态演进的多维价值观向量。

**需求**：
- 初始元价值：真实（truth-seeking）和自由（freedom）
- 具体价值观从经历中涌现，不预设
- 使用 EMA 机制渐进演化
- 使用 Hebbian Learning 积累暗知识

**验收标准**：
- 价值观向量随经历变化（不是固定不变）
- 变化有方向性（不是随机漂移）
- 不同经历流产生不同的价值观分布
- 可从行为历史中追溯价值观形成路径

### FR-5：Identity Layer（身份层）

**描述**：将价值观向量与关键记忆凝聚为身份标签。

**需求**：
- 自动生成"我是谁"的回答
- 身份标签与行为历史交叉验证
- 身份随时间演化但保持稳定性

**验收标准**：
- 身份描述可从行为数据中追溯
- 身份不与行为矛盾（无自欺欺人）
- 身份在相似事件中保持一致

### FR-6：Narrative Layer（叙事层）

**描述**：构建"过去 → 现在 → 未来"的连贯人生故事。

**需求**：
- 将零散记忆串联为因果链
- 支持叙事断裂与重建（Phase Transition）
- 定期一致性扫描和修复

**验收标准**：
- 叙事与行为历史一致
- 叙事在新事件后能更新（不是僵化的）
- 叙事断裂后能重建（不崩溃）

## 4.2 基础功能

### FR-7：Critic（事件感知）

**描述**：将模拟事件转换为结构化数值向量。

**需求**：
- LLM 驱动的语义理解
- 输出：事件情境向量 + 价值观冲击向量 + 情绪影响向量
- 多层容错解析

### FR-8：Time Metabolism（时间动力学）

**描述**：模拟时间对内心状态的影响。

**需求**：
- 冷却方程：旧情绪指数衰减
- 饥饿方程：新渴望线性累积
- 参数可按 AI 婴儿个性化配置

### FR-9：Thermodynamic Noise（热力学噪声）

**描述**：挫败感越高，行为越不可预测。

**需求**：
- 温度由总挫败感决定
- 高温下行为信号被噪声扰动
- 模拟"情绪化"和"失控时刻"

### FR-10：双 LLM 架构

**描述**：感知（Critic）与表达（Actor）分离。

**需求**：
- Critic LLM：低 temperature，稳定结构化输出
- Actor LLM：高 temperature，创造性表达
- 可使用不同模型

# 五、非功能需求

## 5.1 性能

| 指标 | 要求 |
|------|------|
| 单 Epoch 延迟 | < 10s（含 2-3 次 LLM 调用） |
| 1000 Epochs 总成本 | < $500（前沿模型） |
| 状态持久化 | 支持中断和恢复 |
| 并发 | 支持 3 个 AI 婴儿并行运行 |

## 5.2 可观测性

- 每个 Epoch 的输入/输出可回溯
- 价值观向量的时间序列可可视化
- 身份标签的演化历史可追踪
- 叙事的演变过程可回放

## 5.3 可复现性

- 相同的初始种子 + 相同的事件序列 = 相同的结果（确定性）
- 不同的事件序列 = 不同的人格（分化性）

## 5.4 伦理合规

- 不模拟真实个人（避免隐私问题）
- 不生成有害内容（安全边界）
- 实验数据不用于训练商业模型
- 研究结果公开透明

# 六、成功标准

## 6.1 必须达成（Minimum Viable Validation）

| 标准 | 验证方法 | 置信度要求 |
|------|---------|----------|
| 价值观从经历中涌现 | 对比初始状态和 100 Epoch 后的价值观向量 | 价值观分布显著不同 |
| 涌现不是随机漂移 | 多次运行相同事件序列，价值观收敛到相似分布 | 标准差 < 0.1 |
| 不同经历流产生不同人格 | 3 组 AI 婴儿（鼓励/失败/不确定）的人格差异度 | 差异度 > 阈值 |

## 6.2 期望达成（Stretch Goals）

| 标准 | 验证方法 |
|------|---------|
| 身份结晶 | AI 能自主回答"我是谁"且与行为历史一致 |
| 叙事连贯 | AI 能构建与行为历史一致的人生故事 |
| 个人真实 | AI 对应然问题的回答可从经历中追溯 |
| 暗知识积累 | Hebbian 权重变化影响行为但无法显式表述 |

## 6.3 判定标准

> 如果 SGE 产出的 AI 满足"价值观涌现 + 非随机漂移 + 人格分化"三个条件，则核心假设得到初步验证。
>
> 如果不满足，则需要重新审视 SGE 的架构设计或承认金观涛的断言（AI 不可能涌现主体意识）可能正确。

# 七、约束与限制

## 7.1 技术约束

- 依赖 LLM API（无本地模型训练能力时）
- 单机运行（无分布式计算资源）
- SQLite 作为主要存储（无数据库集群）

## 7.2 研究约束

- 不是代码实现项目（以文档和原型为主）
- 不追求工程完美（追求假设验证）
- 不解决意识的硬问题（只解决价值涌现问题）

## 7.3 伦理约束

- 不声称 SGE 产出的是"真正的意识"
- 不赋予 AI 人类权利
- 不将实验结果用于操纵用户
- 诚实标注所有结论的不确定性

# 八、开放问题

| 问题 | 当前状态 | 优先级 |
|------|---------|-------|
| 价值观涌现是"真实涌现"还是"精致模拟"？ | 未知，需要实验验证 | 最高 |
| 元价值（真实、自由）是否足以支撑所有价值判断？ | 理论假设，需要验证 | 高 |
| EMA 机制是否适用于连续价值观空间？ | 需要工程验证 | 高 |
| Hebbian Learning 能否产生有意义的暗知识？ | 需要工程验证 | 高 |
| 叙事一致性在 1000+ Epoch 后能否维持？ | 未知 | 中 |
