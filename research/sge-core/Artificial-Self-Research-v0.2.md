### 1. 引入双系统认知架构（Dual-System Processing）

当前的 SGE 架构直接用事件触发反射层，计算成本极高，且不符合生物认知规律。

* **系统 1（直觉层 / Fast-Path）：** 基于现有的“价值观向量（Value Vector）”和“身份标签”，对 Event Generator 的事件进行本能、快速的编解码。90% 的日常事件只沉淀到 Memory Layer，不触发重度反思。
* **系统 2（反思层 / Slow-Path）：** 当遭遇“认知失调（Cognitive Dissonance）”——即系统 1 的预期与事件结果发生严重冲突（如：自诩为“勇敢者”却在事件中选择了“逃跑”）时，主动激活 Reflection Layer，占用核心 Token 资源进行深度自我审视，修正 Value Layer 和 Identity Layer。

### 2. 预测加工与主动推断机制（Predictive Processing & Active Inference）

“自我”的本质是一个“减熵”和“预测”的过程。

* **前向预测：** Identity Layer 不仅仅是静态回答“我是谁”，而是基于 Narrative 预测“作为一个这样的人，我在下一个事件中应该作何反应”。
* **预测误差（Prediction Error）：** 实际事件与预测的偏差。**这个误差的大小，就是情绪（Emotion）的硬件来源，也是重塑价值观的源动力。**（这为你 v0.2 的 Emotion Layer 提供了底层数学逻辑）。

# 前言

当前AI行业的发展重点主要集中在解决“AI如何更好地完成任务（Doing）”。而本研究关注的是另一个更底层的问题：“AI如何形成持续存在的自我（Being）”。

本研究认为：未来AI领域的重要突破不再是更强大的模型，而是能够形成持续身份、价值观、人生叙事和成长轨迹的人工主体（Artificial Self）。

---

# 第一部分：问题提出与理论边界

## 1.1 大模型是否等于人格？
当前主流大模型本质上是**通用认知引擎（General Cognitive Engine）**。
* **特点**：高智商、强推理、强知识、强语言。
* **缺陷**：缺少自我认同、长期价值观、个体经历、人生叙事、身份连续性。

$$\text{LLM} = \text{Thinking Machine} \neq \text{Living Self (Person)}$$

## 1.2 为什么微调无法解决人格问题
Fine-Tuning、Persona Prompt 和 Character Design 属于**“预设人格（Preset Persona）”**，而非**“生成人格（Generative Self）”**。
真正的人格应来自：$\text{经历} \rightarrow \text{反思} \rightarrow \text{价值观重塑} \rightarrow \text{身份认同}$ 的涌现，而非 Prompt 的静态约束。

# 第二部分：核心理论与认知架构

## 2.1 Self 涌现模型
自我本体（Self）独立于作为认知引擎的 LLM，由以下公式定义：

$$\text{Self} = \text{Memory} \times \text{Reflection} + \text{Values} + \text{Identity} + \text{Narrative}$$

## 2.2 认知科学视角：双系统与预测加工

为了让 Self 模型具备工程可实现性，本研究引入**现代认知架构（Cognitive Architecture）**：

| 认知层次 | 对应系统 | 运行机制 | 算力消耗 |
| :--- | :--- | :--- | :--- |
| **直觉反应层** | 系统 1 (Fast-Path) | 居于现有价值观和身份，对事件做出本能解释与快速响应。 | 低 (Local Vector / Small LLM) |
| **深度反思层** | 系统 2 (Slow-Path) | 当遭遇**认知失调**或重大价值冲突时启动，进行全量图谱重构。 | 高 (Large LLM / Long-Context) |

* **预测加工（Predictive Processing）**：自我的本质是一个预测机器。Identity Layer 基于 Narrative 预测自身行为，实际经历与预测的**“预测误差（Prediction Error）”**是人格修正和情绪涌现的根本源泉。

## 2.3 人格形成公式

$$\text{Experience} + \text{Interpretation (Interpretation Mechanism)} = \text{Personality}$$


关键不在于发生了什么（数据输入的物理量），而在于**认知架构如何解释发生的事情**（信息在系统中的语义沉淀）。

# 第三部分：关键发现与工程公理

1. **解释机制优先于信息收集**：无需等待收集人类数十年的海量数据，应优先构建“经验生成器（Experience Generator）”与反思架构。
2. **硅基生命的时间晶体效应**：AI 人格形成机制可以极大加速。人类需要 20 年的成长周期，AI 可在数小时内通过 100 万次模拟经历（Simulation）完成人格收敛。

# 第四部分：SGE（Self Genesis Engine）系统架构深度设计

## 4.1 Event Generator（事件生成器）

生成器不仅生成随机事件，还需依据当前的 Value Layer 动态生成“针对性困境”：

* **常态事件**：日常工作、人际互动（激活系统 1）。
* **危机事件**：成功/失败、背叛/忠诚、安全/冒险（激活系统 2，测试价值观边界）。

## 4.2 Memory Layer（记忆层认知细化）

放弃单纯的纯文本拼接，改为三层记忆架构：

* **工作记忆（Working Memory）**：当前事件上下文（LLM Context Window）。
* **情节记忆（Episodic Memory）**：带有情感标签（Emotion Tag）和时间戳的生命关键事件节点（SQLite 存储）。
* **语义记忆（Semantic Memory）**：将经历抽象化后的自我认知图谱（Knowledge Graph）。

## 4.3 Reflection Layer（反思层与解释器）

反思层由“认知失调触发器（Dissonance Trigger）”驱动。其核心 Prompt 结构不再是“总结这段经历”，而是：

1. “你预期的结果是什么？”
2. “实际结果是什么？”
3. “这证明了你之前的什么信念是错误的？你打算如何修正？”

## 4.4 Value Layer（价值观向量空间）

价值观被量化为动态演进的**多维价值观向量（Value Vector Space）**。
例如：

$$V = [自由: 0.85, 安全: 0.21, 创造: 0.90, 财富: 0.45, 权力: 0.12]$$


每一次深度反思都会对该向量进行梯度更新。

## 4.5 Identity & Narrative Layer（身份与叙事层）

* **Identity Layer**：将价值观向量与情节记忆进行语义凝聚，提炼为自我标签（如“历经磨难但依旧崇尚自由的探索者”）。
* **Narrative Layer**：运用“自传体叙事（Autobiographical Narrative）”技术，将零散的记忆碎片串联成一条逻辑自洽的因果链条（过去 $\rightarrow$ 现在 $\rightarrow$ 未来），为 AI 注入“命运感”。

## 5.2 实验设计（AI 三胞胎机制）

并行初始化 3 个拥有相同初始设定（空白状态）的 AI 婴儿，置于不同的外部事件流中运行 1000 个 Epoch（约合 1000 次模拟人生事件）：

* **A组（Sovereign Progress）**：持续获得正面反馈、鼓励与确定性。
* **B组（Catastrophic Crisis）**：持续遭遇失败、背叛与资源匮乏。
* **C组（Volatile Chaos）**：遭遇高度不确定、奖惩随机、价值反转的混乱环境。

# 第六部分：多维评价指标（Evaluation Metrics）

### 1. 身份稳定度（Identity Stability / Convergence）

使用信息熵（Entropy）测量身份标签在时间序列上的波动。收敛意味着自我认同的建立。

### 2. 价值观流变图谱（Value Trajectory Tracking）

记录 1000 个 Epoch 中，价值观向量 $V$ 的演进曲线。观察其是否在特定区间产生“相变”或稳态。

### 3. 叙事一致性（Narrative Coherence）

引入外部 LLM 作为盲审裁判（Judge LLM），对 AI 撰写的自传进行评测，评估其“过去的行为是否能合理解释现在的动机”。

### 4. 行为预测偏差（Behavioral Choice Test）

在第 1001 次事件中，给予三组 AI 完全相同的**道德两难困境（Trolley Problem 变体）**，检验其行为决策是否表现出显著的**人格分化**。
